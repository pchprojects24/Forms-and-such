B. Braun Infusomat® Space

Volumetric infusion pump (B. Braun Space system)

What this page is for:
  • First-time orientation + step-by-step use (someone who has never touched the pump before).
  • Practical “what to do” guidance with common errors and what they look like on the device.

This does not replace your site policy, training, or the manufacturer IFU. If anything conflicts: follow local policy + the IFU.

Safety baseline (read once): This pump is designed for Infusomat Space dedicated pump sets. Using unapproved tubing/sets risks free-flow protection, accuracy, and alarm performance. If you are not trained/authorized to operate infusion pumps in your environment, stop and get help.
1) Orientation (what you’re looking at, before you press anything)

Identify the key controls on the front panel

  • Screen (left) — shows infusion status, settings, alarms.
  • Navigation cluster (center) — arrow keys + OK to move through menus and confirm.
  • Start/Stop button (right side cluster) — labeled Start/Stop.
  • Power button (⏻ icon, right side cluster) — turns the pump on/off.
  • Door / tubing channel — where the pump set is routed and secured (free-flow protection depends on correct loading).

What you should verify immediately

  • You have the correct device (Infusomat Space, not a syringe pump).
  • You have the correct pump set (approved Infusomat Space pump set; in date; packaging intact if required by policy).
  • The pump is either on a SpaceStation/docking system or can run on internal battery.
Do not “just start pressing buttons” until tubing is loaded correctly and you are ready to program a verified order.
2) Power & readiness check (get to a safe starting state)
  1. Connect to power if available. If docked on a SpaceStation, confirm it is seated properly.
  2. Turn the pump on: press the ⏻ Power button (right cluster).
  3. Wait for self-test to complete. Watch the screen for completion and any prompts.
  4. Confirm you are in the correct profile/unit configuration (site-specific). If the device shows an unexpected profile/library, stop and escalate per local policy.
  5. Check battery/AC status icon on screen if you expect transport use.
If you see an alarm immediately (before tubing is loaded): don’t ignore it. Read the on-screen message, correct the cause (often door/tubing state/config), or swap the device and report to biomed if needed.
Good state to proceed: device powered, no active alarms, correct profile/library (if applicable), ready/standby screen visible.
3) Tubing / line loading (highest-risk step — do it deliberately)

Before loading

  • Confirm the order and the med/solution, concentration, and route per policy.
  • Use an approved Infusomat Space pump set (in date; correct type for therapy).
  • Prime the line per local policy (method varies by unit/therapy).

Loading sequence (generic but explicit)

  1. Stop/Standby: ensure infusion is not running (press Start/Stop if needed and confirm it is stopped).
  2. Open the pump door (use the door latch/handle as designed; do not force).
  3. Route the tubing exactly along the molded guides inside the door/channel.
    • Follow the printed/embossed routing diagram on the pump.
    • Ensure the tubing sits fully in the channel (no twists; no riding on the edge).
    • Ensure any required clamp/sensor segments are positioned as indicated by the device’s guides.
  4. Confirm free-flow protection/clamp behavior visually as you close the door (your set + correct routing is what makes this protection work).
  5. Close the door firmly until it latches. If it won’t close easily, stop and re-seat tubing—do not force it.
Common “line won’t load / door won’t close” causes: tubing not seated in the guide, wrong set type, a clamp segment positioned incorrectly, or tubing caught at the hinge/latch edge.
If the pump alarms after loading: read the exact message and re-check (1) door fully latched, (2) tubing seated and routed correctly, (3) upstream clamp/roller clamp open, (4) no kinks.
4) Programming the infusion (enter parameters and verify them)

Exact menus vary by configuration (drug library, lock levels, profiles). The safe pattern is always: select correct mode → enter required parameters → confirm summary → start → verify running screen.

Button-level basics

  • Arrow keys move between fields/menus.
  • OK confirms a selection/entry.
  • Start/Stop starts or pauses/stops infusion (depending on configuration/prompt).

Programming workflow

  1. Select the therapy/mode required at your site (basic infusion vs drug library dose mode, if available).
  2. Enter ordered values using the navigation keys:
    • Commonly the pump expects 2 of 3: Rate, VTBI, Time (and calculates the third).
    • In drug library modes, you may be entering dose-based parameters (site-dependent).
  3. Stop and read the confirmation/summary screen before starting:
    • Verify: patient (if used), drug/solution, concentration (if applicable), rate, VTBI/time, line route/connection plan.
    • If anything looks wrong: back out and correct before starting.
High-consequence error pattern: entering the “right numbers in the wrong units/mode.” If your site uses drug libraries/profiles, confirm you are in the correct profile before trusting limits/prompts.
5) Connect to patient & start infusion (verify it is actually running correctly)
  1. Trace the line end-to-end (bag/syringe → pump → patient connection point) per policy.
  2. Confirm patient access (correct line/port; patency checks per policy).
  3. Connect to the patient and ensure any required clamps are in the correct position for infusion.
  4. Press Start/Stop to start infusion (confirm any prompt shown).
  5. Immediately verify on the run screen:
    • Status shows running (not standby/stopped).
    • Rate/VTBI/time reflect what you intended.
    • No active alarm banner is present.
  6. Observe the first 30–60 seconds for occlusion/air/door/line alarms and correct as needed.
If an occlusion alarm triggers right away: think “clamp closed / kink / wrong port / upstream issue” before assuming a device fault.
6) Pause / stop / end (and unload safely)
  1. Pause/Stop: press Start/Stop and follow on-screen prompt(s) (site behavior varies).
  2. Clamp at the patient if required by your unit policy before disconnecting/changing the set.
  3. Unload: open the door and remove tubing carefully without forcing any components.
  4. End-of-therapy: document per policy; dispose/retain the set as required.
Transport reminder: if moving the patient, confirm battery status and secure the pump/lines to prevent accidental pull or kinking.
Common pitfalls (what goes wrong most often)
  • Wrong tubing/set (not an approved Infusomat Space pump set) → unreliable operation / alarms / safety feature failures.
  • Door won’t close → tubing not seated in the channel, routed incorrectly, or wrong set type.
  • Immediate occlusion alarms → upstream clamp closed, kinked tubing, wrong patient port, downstream obstruction.
  • Upstream occlusion / closed clamp → roller clamp left closed, bag/line issue before pump.
  • Parameter errors → wrong mode/profile, wrong units, skipped summary verification.
  • Assuming “Start” means running → actually still in standby/stopped due to a prompt or alarm.
Do’s & Don’ts (short safety rules)

Do

  • Do use only approved Infusomat Space pump sets (in date).
  • Do trace the line end-to-end before starting.
  • Do verify the run screen after pressing Start/Stop.
  • Do stop and read alarm text — fix the cause, don’t guess.
  • Do confirm you’re in the correct profile/library (site-specific).

Don’t

  • Don’t force the door closed.
  • Don’t bypass routing guides or clamp mechanisms.
  • Don’t “copy settings” from memory without verifying mode/units.
  • Don’t ignore upstream clamps/roller clamps when troubleshooting occlusion alarms.
  • Don’t use unapproved tubing/sets.
